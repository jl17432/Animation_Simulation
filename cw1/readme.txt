The project contains the code for:

1. reading and rendering a set of motion in a BVH file
2. write a set of motion into the bvh file
3. some attempts on the IK handle


UI:
to load a bvh file, press 'L' on the keyboard

to write a bvh file, press 'W' on the keyboard

to pause the animation, press 'p' on the keyboard

to move to the next frame, press '=' on the keyboard

to move to the previous frame, press '-' on the keyboard

left-click and drag to zoom-in/zoom-out

right-click and drag to rotate the camera
